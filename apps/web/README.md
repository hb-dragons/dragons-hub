# @dragons/web

Next.js frontend app for the Dragons workspace.

## Development

```bash
pnpm install
pnpm --filter @dragons/web dev
```

Open `http://localhost:3000`.

## Commands

- `pnpm --filter @dragons/web lint`
- `pnpm --filter @dragons/web typecheck`
- `pnpm --filter @dragons/web build`
- `pnpm --filter @dragons/web start`

This app imports UI components and global styles from `@dragons/ui`.
